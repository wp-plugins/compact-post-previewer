Compact Post Previewer will allow you to display your posts in a short area using shortcode.
== Description ==
Compact Post Previewer  will allow user to display posts and custom post types in shorten way and can help the visitor of site to access desired posts in one page.

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Compact Post Previewer' and install it
3. Go to Dashboard > Compact Post Previewer.Save options
4. Get shortcode using Get Shortcode Button
5. Use shorcode in post and pages and see plugin in action
